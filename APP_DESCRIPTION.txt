PRODUCT SNAPSHOT
----------------
NextGen Smart Finance Manager is a full-stack personal finance application for
tracking income, expenses, budgets, savings goals, subscriptions, and tax
planning in one place.

The platform combines a Django REST backend with a static frontend built in
vanilla HTML/CSS/JavaScript. Insights are generated by deterministic,
rule-based analytics modules (no external LLM dependency), making behavior
predictable, private, and easy to maintain.

The product is built for Indian users first (INR defaults, local merchant
keyword coverage, UPI/cash/bank workflows), while also supporting user-level
currency preferences such as INR, USD, EUR, and GBP.

TECH STACK
----------
  Backend      : Django 6.x, Django REST Framework
  Frontend     : Vanilla HTML5, CSS3, JavaScript (ES5), Chart.js 4.x
  Database     : MySQL (configured in Django settings)
  Auth         : JWT (HS256, 24-hour tokens) + Google OAuth 2.0
  Payments     : Razorpay Orders API + signature verification
  Cache        : Redis (django-redis with local-memory fallback)
  Async Jobs   : Celery + Redis broker/result backend
  Config       : python-dotenv for environment variables
  CORS         : django-cors-headers

AI / INSIGHTS ENGINE (RULE-BASED)
---------------------------------
Module                  Description
------                  -----------
Insights Engine         Multi-signal analysis covering spending trends,
                        budget pressure, category deltas, and savings behavior

Auto-Categorizer        Keyword-driven classifier with Indian merchant focus
                        and user keyword overrides

Receipt Scanner         Regex-based parser for amount/date/merchant extraction
                        from pasted receipt text

Health Score            Weighted score across savings, budget adherence,
                        expense ratio, consistency, and goal progress

Subscription Detector   Detects repeating transactions over historical windows

Goal Planner            Computes required monthly contribution and shortfall

What-If Simulator       Projects outcomes from category spending adjustments

Behavioral Engine       Detects habits such as weekend spikes and payday splurge

Budget Alerts           Warns users at configurable threshold milestones

CSV Importer            Auto-column mapping, date normalization, staged import

SETUP AND RUNNING
-----------------
Backend:
  cd backend
  pip install django djangorestframework django-cors-headers PyJWT
  pip install python-dotenv PyMySQL google-auth requests
  pip install celery redis django-redis
  python manage.py makemigrations
  python manage.py migrate
  python manage.py create_defaults
  python manage.py runserver 8000

Background workers (separate terminals):
  cd backend
  celery -A smartfinance worker -l info
  celery -A smartfinance beat -l info

Redis:
  Run Redis locally (default URL: redis://127.0.0.1:6379)

Frontend:
  cd frontend
  python -m http.server 5500

Default local URLs:
  Backend  : http://127.0.0.1:8000
  Frontend : http://127.0.0.1:5500


DESIGN AND ENGINEERING PHILOSOPHY
---------------------------------
  - Rule-based AI modules for transparency and deterministic behavior
  - India-first product framing with practical daily money workflows
  - Progressive premium model without blocking core budgeting fundamentals
  - Design consistency across landing and in-app experiences
  - Clean API boundaries between auth, finance, insights, and tax domains
  - Service-first backend organization for easier testing and scale
  - Security-first auth lifecycle with refresh-token rotation and auditing
